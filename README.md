# sweet-log

> 将 log 展示在页面上，便于调试。

# 引入方式

```html
<script src="http://cdn.jsdelivr.net/npm/sweet-log/dist/index.js"></script>
```

# 配置

```js
window.SWEET_LOG_CONFIG = {
  /** 是否默认展示 */
  display: false,
  /** 位置，支持 top / right / bottom / left */
  position: 'right',
  /** 展示区域宽度 */
  width: '',
  /** 展示区域高度 */
  height: '',
}
```
